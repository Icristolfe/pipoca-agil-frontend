<h1 align="center"> Pipoca Ágil React + Vite + Yarn </h1>

## :checkered_flag: Getting Started ##

```bash
# Clonar projeto
$ git clone git@github.com:Icristolfe/NeoApp-test.git

#no terminal

##Instale o Yarn
$ npm install -g yarn

# Acesso
$ cd pipoca-agil

# instale as dependencias
$ yarn  

# Rode o projeto 
$ yarn start  

# O server será iniciado no localhost
```
